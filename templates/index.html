What is Microsub?

Microsub is a draft standard specification that separates the roll of a feed reader and a feed server. Using the Microsub standard, feed readers and servers can be developed independently. This means that users get more choice in terms of the application that they use to look at their feeds.

James, the creator of this tool, uses this server to subscribe to and manage his feeds. He used to use clients such as Monocle (web) and Indigenous (iOS) to monitor his feeds. He recently implemented his own client on this server that he can use to read his feeds.

How can I learn more about Micropub?

The IndieWeb wiki contains the information you'll need to have about Micropub to get started. The official editor's draft of the specification is also available on the IndieWeb wiki.

Some useful resources to look at are the following pages:

  • Microsub
  • Microsub Clients
  • Microsub Servers
  • Microsub Specification

Can I host my own Micropub client and server?

Yes! You can host your own Microsub endpoint. This client and server is open-sourced so you can use this one if you would like. This is the ideal configuration if you choose to use a static generator like Jekyll on your site. You can deploy a version of this Micropub client and server by going to the project GitHub page and following the instructions.

You can find other Micropub clients and servers on the IndieWeb wiki.
