How to use this tool

If you already have a Microsub server set up and connected to your website, you can go ahead and use the reader component of this service immediately. All you have to do is click "Login" and sign in with your domain name.

If you do not have a Microsub server set up, don't worry. You can use this site as a server!

To do so, add the following HTML tag to the head tag on your website home page:

<link rel="microsub" href="https://microsub.jamesg.blog/endpoint">

That tag will let this service know you are using microsub.jamesg.blog as a Microsub server.

Once you have added the tag, you are ready to sign in! Click "Login" in the navigation bar to get started.
